The Vic-20

Foreword:

In this page, you will learn about how to program the Vic-20. It is a very interesting machine, because almost every aspect of it's operation is handled directly by the user through commands. Most modern computers have a user friendly layer that protects you from all the complicated parts, which makes it easier to use; however, there is a great value to understanding computers at the lower level, and its just plain fun. It might sound somewhat intimidating, but it's not as complicated as its wrapped up to be, so let's begin!

Getting Startted

In order to program a Vic-20, you will need one, which is unfortunatly very expensive; however, you don't neccessarily need a real Vic-20. at mdawson.net, there is an emulator that allows you to use a virtual vic 20 online! You can also check out the original manual, which has great explainations for how the machine works.
